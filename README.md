# Faculty Availability Status System

## Overview
QR-enabled web system for real-time faculty availability display using Google Forms, Sheets, and GitHub Pages.

## Features
- Faculty update availability via Google Forms
- Students view live dashboard via QR code
- Shows cabin number, status, free slots, last updated

## Live URLs
**Faculty View:**  
`https://ashishmerit.github.io/Faculty-Availability-Status-/Faculty_view.html`

**Student Dashboard:**  
`https://ashishmerit.github.io/Faculty-Availability-Status-/student_view.html`

## Usage
1. **Faculty:** Scan Faculty QR → Update status
2. **Students:** Scan Student QR → View availability

## Files
- `Faculty_view.html` - Faculty entry page
- `student_view.html` - Student dashboard

## Setup
1. Update Google Forms/Sheets CSV URLs in HTML files
2. Enable GitHub Pages (Settings → Pages → main branch)
3. Generate QR codes with the URLs above
4. Print & display QR codes

## Tech Stack
- HTML/CSS/JavaScript
- Google Forms & Sheets
- GitHub Pages hosting
- QR Codes
